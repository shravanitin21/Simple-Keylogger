# Python Keylogger (Educational Project)

A beginner-level Python project to understand keyboard event handling using the `pynput` library. 
It logs typed characters into a text file while ignoring special keys.

## Features
- Captures keyboard input
- Handles space and enter keys
- Ignores special keys
- Stores output in a text file

## Technologies Used
- Python
- pynput library

## Disclaimer
This project is created strictly for educational purposes.
